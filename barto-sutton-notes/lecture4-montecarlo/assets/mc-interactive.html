Returns Table — G values collected per (state, action) pair

MC Process: ① Generate complete episodes → ② Compute return G backwards from terminal for each visited (s,a) → ③ Collect G values across all episodes → ④ Q(s,a) = average of all G values for that (s,a) pair → ⑤ Extract greedy policy: π(s) = argmaxa Q(s,a). The key insight: each episode gives one G sample — only averaging over many episodes gives reliable Q estimates.
