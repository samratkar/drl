1. $G_0 = R_1 + R_2 + R_3 = 2 + (-1) + 5 = 6$.
2. $G_0 = R_1 + \gamma R_2 + \gamma^2 R_3 = 2 + 0.5(-1) + 0.25(5) = 2 - 0.5 + 1.25 = 2.75$.
3. $G_t = R / (1 - \gamma) = 10 / (1 - 0.9) = 10 / 0.1 = 100$.
4. $r(s, a) = 0.3(5) + 0.7(-2) = 1.5 - 1.4 = 0.1$.
5. $p(s_1 \mid s, a) = \sum_r p(s_1, r \mid s, a) = 0.3$.
6. $V(S) = \sum_a \pi(a\mid s) [R + \gamma V(s')] = 0.25 [0 + 0.9(10 + 2 + 0 + 5)] = 0.25 [0.9(17)] = 0.25 [15.3] = 3.825$.
7. $q_\pi(s, a) = R + \gamma V(s') = 10 + 0.9(20) = 10 + 18 = 28$.
8. $V(s) = 0.2[1 + 1(0)] + 0.8[10 + 1(5)] = 0.2[1] + 0.8[15] = 0.2 + 12 = 12.2$.
9. $V(s) = \max \{ 0 + 0.9(10), 6 + 0.9(5) \} = \max \{ 9, 6 + 4.5 \} = \max \{ 9, 10.5 \} = 10.5$.

12. $q(\text{high, search}) = 0.6[3 + 0.8(10)] + 0.4[3 + 0.8(5)] = 0.6[11] + 0.4[7] = 6.6 + 2.8 = 9.4$.
13. $V(50) = 0.4[V(100)] + 0.6[V(0)] = 0.4(1) + 0.6(0) = 0.4$.
14. $V(A) = 1 + 0.5 V(B)$; $V(B) = 1 + 0.5 V(A)$.
    $V(A) = 1 + 0.5(1 + 0.5 V(A)) = 1 + 0.5 + 0.25 V(A) \Rightarrow 0.75 V(A) = 1.5 \Rightarrow V(A) = 2$. By symmetry, $V(B) = 2$.
15. $a_1: 0.5[10 + 0] + 0.5[0 + 10] = 5 + 5 = 10$.

16. $\gamma^k < 0.1 \Rightarrow 0.9^k < 0.1$.
    $k \log(0.9) < \log(0.1) \Rightarrow k (-0.0457) < -1 \Rightarrow k > 1 / 0.0457 \approx 21.8$. So $k=22$.
17. $r(s, a) = \sum_r r \sum_{s'} p(s', r \mid s, a) = 10(0.2) + 0(0.8) = 2$.
18. $V_{\pi'}(s) \ge q_\pi(s, \pi'(s)) = 12$. The lower bound is 12.
