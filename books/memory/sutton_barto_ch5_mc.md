# Chapter 5: Monte Carlo Methods (pp. 91-117)

## Core Idea
Learn from complete episodes of experience. No model needed. No bootstrapping. Estimates for each state are independent.

## MC Prediction
- **First-visit MC** (p. 92): Average returns following first visit to s per episode. Unbiased, i.i.d., converges by law of large numbers.
- **Every-visit MC** (p. 93): Average all returns for s. Biased but converges quadratically. Easier to extend to function approximation.

## MC Control

### MC with Exploring Starts (p. 99)
Assume all state-action pairs have nonzero probability of being selected as episode start. After each episode, update Q(s,a) and make pi greedy w.r.t. Q. Converges to optimal.

### On-policy MC Control (epsilon-soft) (p. 101)
No exploring starts needed. Maintain epsilon-soft policy. Policy improvement: make epsilon-greedy w.r.t. q. Converges to best epsilon-soft policy (v_tilde_*), not true optimal.

### Off-policy MC Control (p. 111)
Learn about target policy pi (greedy) from data generated by behavior policy b (soft). Uses weighted importance sampling. **Limitation**: learns only from tails of episodes consistent with pi — if nongreedy actions are common, learning from early states is very slow.

## Importance Sampling (pp. 103-109)
- **Ratio**: rho_{t:T-1} = product pi(A_k|S_k) / b(A_k|S_k). MDP dynamics cancel.
- **Coverage requirement**: pi(a|s) > 0 => b(a|s) > 0
- **Ordinary IS**: V(s) = (sum rho*G) / count. Unbiased. Can have INFINITE variance (Example 5.5).
- **Weighted IS**: V(s) = (sum rho*G) / (sum rho). Biased (-> 0). Always finite variance. **Strongly preferred**.
- **Incremental weighted IS** (p. 109): V_{n+1} = V_n + (W_n/C_n)[G_n - V_n], C_{n+1} = C_n + W_{n+1}

## Per-Decision Importance Sampling (p. 114)
Apply IS ratios to individual rewards rather than entire returns. Reduces variance by removing unnecessary IS factors whose expectation is 1.

## Key Examples
- **Blackjack** (p. 93): 200 states, MC prediction for "stick on 20/21" policy. MC ES finds optimal matching Thorp's basic strategy.
- **Infinite variance example** (p. 106): One-state MDP where ordinary IS has E[rho^2*G^2] = sum 1.8^k = infinity.

## Key Distinctions
- **MC vs DP**: MC needs no model, doesn't bootstrap, state estimates are independent, can focus on subset of states
- **On-policy vs off-policy**: On-policy evaluates the followed policy. Off-policy separates behavior from target — more general but higher variance.
- MC can't easily be used if episode termination isn't guaranteed for all policies
